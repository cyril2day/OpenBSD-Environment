# OpenBSD-Environment

Personal desktop configurations and files in OpenBSD

## Requisites

- OS: `OpenBSD`
- Session: `xfce4-session`
- Window Manager: `i3`
- Status Bar: `i3status`
- App Launcher: `rofi`
- Fonts: `imported from Mac`, `Windows`
